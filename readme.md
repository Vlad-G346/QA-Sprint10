# Тесты на проверку чек-листа для параметра name при создании набора в Яндекс.Прилавок с помощью API Яндекс.Прилавок.
- Для запуска тестов должны быть установлены пакеты pytest и requests
- Запуск всех тестов выполняется командой pytest

# Ссылка на проект в GitHub
https://github.com/Vlad-G346/Pytest_Project10_Sprint10

# Задача:
Автоматизировать чек-лист проверок для поля name в запросе на создание набора. 

# Подготовка:
1. Сгенерировать и запустить сервер для Яндекс.Прилавок 
2. Поместить ссылку в файл configuration.py в поле URL_SERVICE

# Создание набора для пользователя:
1. Выполнить POST запрос на создание нового пользователя и запомнить токен авторизации authToken.
2. Выполнить POST запрос на создание личного набора для этого пользователя. Обязательно передать заголовок Authorization.

# Основные шаги:
1. Составить функций для проверки предоставленного чек-листа.
2. Запустить все тесты командой pytest
